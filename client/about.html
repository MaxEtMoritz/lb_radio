About <%= title %>

<%= title %> is a utility to submit your radio listening activity to ListenBrainz.
since most dedicated radio receivers (i'm not talking about web radio applications or Websites for listening to Internet radio streams) are dumb devices without internet connection there is no good way to submit listens done with them.
This is where this website kicks in: When turning on the radio receiver, you can search for the radio station on here and click "Start listening". All songs this radio station is playing are now submitted to ListenBrainz until you click "Stop listening" again.

FAQ

Who made this website?

Me, MaxEtMoritz. My GitHub and MusicBrainz / ListenBrainz profiles.

I can't find my favorite radio station via the search!

For a radio station to show up, two things are needed:

  1. The radio station has to offer an Internet radio stream that carries metadata about the song this station is currently broadcasting.
    You can check if your station offers an internet radio stream by visiting their website and looking if there is a way to listen the station on there.
  2. This internet radio stream must be contained in the Radio Browser Database.
    If it is not, you can add it to the database without the need to create an account. just visit the RadioBrowser website and add it. When the internet radio stream was verified to be functional by the RadioBrowser Database, you will be able to find your station here.

Why don't i hear anything when clicking "start listening"?

This is intentional. The purpose of this web site is to submit listens done with dumb, offline radio receivers. If you already listen to the station via a radio receiver, you do not need to hear sound on here, right?
